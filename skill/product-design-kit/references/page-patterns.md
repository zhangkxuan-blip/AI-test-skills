# 页面模式库（Page Patterns）

> 用于 Step 2 线框图生成。匹配页型 → 套用骨架。

## 1. 列表页 (List)

```
┌─────────────────────────────────┐
│ [Logo] 顶部导航      [搜索][用户] │  ← 顶栏
├─────────────────────────────────┤
│ 标题                  [+ 新建]   │  ← 标题栏 + 主操作
│ ┌──[Tab1][Tab2][Tab3]──────┐    │  ← 筛选/Tab
│ │ [筛选] [筛选] [搜索框]    │    │
│ └────────────────────────────┘   │
│ ┌────────────────────────────┐  │
│ │ ☐ 项目 1   状态  时间  ...│  │  ← 列表行
│ │ ☐ 项目 2   状态  时间  ...│  │
│ └────────────────────────────┘  │
│         [< 1 2 3 >]              │  ← 分页
└─────────────────────────────────┘
```

## 2. 详情页 (Detail)

```
┌─────────────────────────────────┐
│ ← 返回   标题            [...]  │
├─────────────────────────────────┤
│ ┌────────────┬─────────────┐    │
│ │  主信息区   │  侧栏元数据  │    │
│ │  内容...    │  · 状态     │    │
│ │  内容...    │  · 时间     │    │
│ │             │  · 负责人   │    │
│ ├────────────┤  · 标签     │    │
│ │  Tab 区     │             │    │
│ │  评论/历史  │             │    │
│ └────────────┴─────────────┘    │
│ [编辑]  [删除]  [分享]           │
└─────────────────────────────────┘
```

## 3. 表单页 (Form)

```
┌─────────────────────────────────┐
│ ← 返回   新建 / 编辑             │
├─────────────────────────────────┤
│ 步骤指示器  ①─②─③                │  ← 多步表单时
│                                  │
│  字段标题 *                      │
│  [输入框                  ]      │
│                                  │
│  下拉                            │
│  [选项 ▾                  ]      │
│                                  │
│  描述                            │
│  [文本域                       ] │
│  [                             ] │
│                                  │
├─────────────────────────────────┤
│         [取消]  [保存草稿] [提交] │
└─────────────────────────────────┘
```

```
┌─────────────────────────────────┐
│ 标题         [时间范围 ▾] [导出] │
├─────────────────────────────────┤
│ ┌──KPI 1─┐┌──KPI 2─┐┌──KPI 3─┐ │
│ │ 1,234  ││ 56%    ││ ↑12%  │ │
│ └────────┘└────────┘└────────┘ │
│ ┌─────────────┬───────────────┐│
│ │  趋势图      │   占比环图    ││
│ │  📈         │   🍩          ││
│ └─────────────┴───────────────┘│
│ ┌──────────────────────────────┐│
│ │  最近活动列表                 ││
│ └──────────────────────────────┘│
└─────────────────────────────────┘
```

## 5. 弹窗 (Modal)

```
       ┌──────────────────┐
       │  标题       [×]  │
       ├──────────────────┤
       │                  │
       │  内容/表单        │
       │                  │
       ├──────────────────┤
       │     [取消] [确定] │
       └──────────────────┘
```

## 6. 抽屉 (Drawer)

```
┌────────────────┬──────────────┐
│                │ 标题      [×]│
│                ├──────────────┤
│   原页面遮罩    │              │

│                │              │
│                │              │
│                │ [操作1][操作2]│

```

## 选型决策

| 我有... | 选用 |
|---|---|
| 多条同结构数据 | 列表页 |
| 单条数据深度查看 | 详情页 |
| 用户输入 | 表单页 |
| 数据汇总/监控 | 仪表盘 |
| 临时确认/编辑 | 弹窗（少量字段）/ 抽屉（多字段） |
| 配置项 | 设置页 |
| 关键词检索 | 搜索结果页 |
