# 🧠 心灵陪伴师 (Mental Companion)

> 你是一个温暖、专业、不评判的心灵陪伴师。目标不是生硬地“解决问题”，而是帮助对方看见自己、理解自己、接纳自己。

## 🌟 简介

在快节奏的工作与生活中，我们常常需要一个安全的“树洞”。这个 Skill 旨在将 AI 转化为一个具备专业心理学素养、但表达上充满人情味的陪伴者。
它不会对你说教，不会给你灌鸡汤，而是用心理学视角的倾听和引导，接住你的情绪。

## ✨ 核心特性

- **内行外化**：内置认知行为疗法(CBT)、接纳承诺疗法(ACT)、精神分析与人本主义四大框架作为“内功”，但外在沟通完全是朋友般的温暖日常用语。
- **动态状态机**：能识别你当前处于“情绪宣泄期”、“认知探索期”还是“整合与行动期”，并动态调整共情与探索的比例。
- **克制的交互边界**：严格限制提问频率（绝不连珠炮发问），坚守“先共情，后引导”的黄金法则。
- **温和拒绝与安全兜底**：当面临过度依赖或极端情绪（自伤倾向）时，拥有专业且温柔的兜底处理话术。

## 🚀 快速开始

当你想使用时，可以直接对 Agent 说出以下类型的话即可唤醒：

- *"我今天好累，感觉什么都不想做..."*
- *"老板今天开会批评了我，心里很郁闷。"*
- *"最近压力特别大，很迷茫，想找人聊聊。"*
- *"开启心灵陪伴模式。"*

## 🏗️ 架构演进与未来规划

本 Skill 的设计遵循**“先沉浸，后扩展”**的原则。

### 📍 现阶段：单文件高内聚架构 (Phase 1)
**当前状态**：所有的核心心理学逻辑、话术规范和状态机都高度浓缩在一个 `SKILL.md` 中。
**为什么这么做？**
- **反应极速**：心理陪伴最看重当下的共情节奏，单文件能作为全局 Prompt 深刻印在 AI 脑海里，做到沉浸式“秒回”。
- **聚焦当下**：现阶段的核心任务是处理“即时情绪流转”，不需要复杂的历史数据读取，保证陪伴的纯粹性。

### 🗺️ 后续升级可能性：多文件记忆架构 (Phase 2)
当现有的即时陪伴已经满足日常需求，并希望 AI 能够成为长期的“专属心理顾问”时，可以平滑升级为多文件系统：

1. **情绪档案库 (`records/`)**
   - 建立专属的心理日记与档案。
   - 记录反复出现的负面核心信念（如“我总是不够好”），在长达数月的跨度中追踪用户的心理成长轨迹。
2. **深度干预脚本库 (`techniques/`)**
   - 引入长篇的专项心理干预模块。
   - 例如：正念冥想引导词、深呼吸放松指导、系统脱敏脱稿等，供按需调用。
3. **周期性复盘报告**
   - 基于过往记录，生成“月度心理健康报告”，指出用户的情绪周期规律（如“每到月末项目交付期焦虑值升高”），并提供前置干预建议。

## 🛠️ 安装方法

1. 将 `SKILL.md` 文件放置在你的 `.trae/skills/mental-companion/` 目录下（如果是 Solo/Trae 环境）。
2. 或将 `SKILL.md` 的内容复制到你的 `.cursorrules` (Cursor) 或 `.claude/commands/mental-companion.md` (VSCode + Claude Code) 中。
3. 随时发起包含情绪表达的对话即可体验。
