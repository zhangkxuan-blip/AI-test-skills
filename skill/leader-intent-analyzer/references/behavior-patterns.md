# 领导行为模式分析参考手册

## 一、决策风格识别

### 1.1 果断型 vs 犹豫型

| 特征 | 果断型领导 | 犹豫型领导 |
|------|-----------|-----------|
| 典型用语 | "就这么定了""立即执行" | "我再想想""先放一放""多看看" |
| 决策速度 | 快，当场给结论 | 慢，需要多次讨论 |
| 变更频率 | 低，定了不轻易改 | 高，经常调整方向 |
| 应对要点 | 快速响应，减少犹豫时间 | 给足安全感，提供多方案对比 |
| 话术建议 | "明白，我今天就出初版" | "我准备了A/B两个方向，您看哪个更贴近您的想法？" |

### 1.2 数据驱动型 vs 直觉型

| 特征 | 数据驱动型 | 直觉型 |
|------|-----------|--------|
| 典型用语 | "数据呢？""有没有依据？""转化率多少？" | "我感觉...""凭经验...""市场就是这样" |
| 汇报重点 | 数字、图表、对比 | 案例、故事、趋势判断 |
| 质疑方式 | 质疑数据来源和逻辑 | 质疑方向感和时机 |
| 应对要点 | 提前准备数据，用数字说话 | 强调市场敏感度和时机把握 |
| 话术建议 | "根据过去3个月数据，这个方案预计提升15%" | "这个方向符合目前行业趋势，竞品X也在做类似尝试" |

### 1.3 集权型 vs 授权型

| 特征 | 集权型 | 授权型 |
|------|--------|--------|
| 典型用语 | "按我说的做""这个我来定" | "你看着办""我相信你的判断" |
| 参与程度 | 深入细节，频繁过问 | 只看结果，过程不干涉 |
| 容错空间 | 低，期望严格执行 | 高，允许试错 |
| 应对要点 | 多请示，关键节点汇报 | 主动承担，定期同步进度即可 |
| 话术建议 | "这个关键点我想和您确认一下方向" | "目前进展顺利，预计周三交付，有问题我随时同步" |

## 二、沟通风格识别

### 2.1 直接型 vs 委婉型

**直接型信号**：
- 消息短，没有客套话
- 批评直接指出问题
- 用"不行""错了"等明确否定
- 群里直接@人提问题

**委婉型信号**：
- 常用"是不是可以...""也许...""建议..."
- 批评用"还有提升空间""再优化一下"
- 不直接说"不"，而是沉默或转移话题
- 喜欢先肯定再提建议

### 2.2 时间敏感度识别

| 时间表达 | 真实含义 | 建议响应时间 |
|---------|---------|------------|
| "现在""马上" | 立刻，放下手头所有事 | 5分钟内响应 |
| "今天" | 下班前 | 2-4小时内给进度 |
| "尽快" | 视优先级，通常1-2天 | 当天内确认收到并给时间计划 |
| "这周" | 周五前 | 周三前给初版 |
| "有空的时候" | 不紧急，但别拖太久 | 3-5天内响应 |
| 没有说时间 | 默认按常规节奏 | 主动询问期望时间 |

### 2.3 情绪状态识别

| 信号 | 可能状态 | 应对策略 |
|------|---------|---------|
| 消息带感叹号/多个问号 | 焦虑或不满 | 先安抚，再给确定性 |
| 深夜/周末发工作消息 | 压力大或焦虑 | 及时响应表示重视，但不必立即交付 |
| 已读不回 | 犹豫/不满/忙 | 24小时后礼貌跟进 |
| 突然变得很客气 | 可能有坏消息或要裁人 | 提高警惕，准备预案 |
| 频繁发语音/打电话 | 紧急或情绪强烈 | 立即接听，先听少说 |
| 消息变短、变少 | 失望或失去兴趣 | 主动同步进展，重建信任 |

## 三、优先级与价值观识别

### 3.1 关键词频率分析

记录领导在不同场合高频提到的词汇，这些往往代表其核心价值观：

- **"用户体验""用户价值"** → 用户导向型，决策以用户利益为先
- **"数据""增长""转化"** → 结果导向型，看重可量化的产出
- **"合规""安全""风险"** → 稳健型，宁可慢不可错
- **"创新""突破""颠覆"** → 进取型，愿意承担风险试错
- **"效率""成本""ROI"** → 务实型，关注投入产出比
- **"团队""协作""文化"** → 人文型，重视组织氛围和人才培养

### 3.2 利益权衡逻辑

观察领导在资源冲突时的选择倾向：

- **速度 vs 质量**：赶工期时牺牲哪个？
- **短期 vs 长期**：更关注季度业绩还是长期布局？
- **用户 vs 商业**：用户利益和商业利益冲突时偏向哪边？
- **创新 vs 稳定**：更愿意尝试新方案还是沿用成熟方案？

## 四、人际关系倾向

### 4.1 向上管理风格

| 类型 | 特征 | 对你意味着什么 |
|------|------|--------------|
| 报喜不报忧型 | 只向上汇报好消息，问题自己扛 | 领导可能不了解真实困难，你需要适度同步风险 |
| 频繁求助型 | 经常找上级要资源、要决策 | 领导可能权限有限，重大决策需要更上层支持 |
| 对抗型 | 经常和上级意见不合，公开争论 | 注意站队，避免被卷入政治斗争 |
| 依附型 | 对上级言听计从，没有独立判断 | 领导可能是执行者而非决策者，关键决策要追溯到更高层 |

### 4.2 对下属的信任程度

**高信任信号**：
- 授权范围大，不干涉过程
- 让你代表他参加会议
- 分享敏感信息（如人事变动、战略方向）
- 犯错时保护你，对外承担责任

**低信任信号**：
- 频繁追问进度细节
- 要求所有决策都要他确认
- 不透露背景信息，只给指令
- 犯错时当众批评或甩锅

## 五、常见领导类型速查表

### 类型A：老虎型（目标导向、强势果断）
- **特征**：说一不二，重结果轻过程，不耐烦解释
- **沟通要点**：开门见山，先说结论，用数据支撑
- **禁忌**：不要铺垫太长、不要讲困难、不要问"为什么"
- **话术模板**："这个任务我计划在X日完成，预计产出Y，需要您确认Z"

### 类型B：孔雀型（外向热情、喜欢表达）
- **特征**：话多，喜欢头脑风暴，想法多变
- **沟通要点**：多互动，给足情绪价值，把他的想法落地
- **禁忌**：不要冷场、不要只发文字不表态、不要否定他的创意
- **话术模板**："这个想法很有启发！我梳理一下，您看是不是这个意思..."

### 类型C：考拉型（温和稳重、注重关系）
- **特征**：好说话，不施压，但决策慢
- **沟通要点**：建立信任关系，给足安全感，帮他做决策
- **禁忌**：不要催太紧、不要突然变卦、不要在公开场合让他难堪
- **话术模板**："这个方向我和团队讨论过，大家都觉得可行，您看我们按这个计划推进可以吗？"

### 类型D：猫头鹰型（严谨细致、追求完美）
- **特征**：抠细节，要数据，反复确认
- **沟通要点**：准备充分，细节到位，逻辑清晰
- **禁忌**：不要模糊表述、不要想当然、不要提交粗糙的产出
- **话术模板**："这个方案我准备了详细的数据支撑，包括A、B、C三个维度，您看还有哪里需要补充？"

### 类型E：变色龙型（适应性强、难以捉摸）
- **特征**：因时因地变化，没有固定模式
- **沟通要点**：灵活应对，多观察少判断，随时准备调整
- **禁忌**：不要贴标签、不要假设他上次的态度这次也一样
- **话术模板**："我先按这个方向准备一版，您随时有调整我随时配合"
